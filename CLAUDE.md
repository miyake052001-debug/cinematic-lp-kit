# Cinematic LP Kit — Claude Code 指示書

このフォルダは「スクロールで世界が変わるシネマティックWebGLランディングページ」のテンプレートキット。
**あなた（Claude Code）の仕事は、ユーザーのブランド設計をもとに `index.html` の CONFIG ブロックだけを書き換えて、そのブランド専用のLPを完成させること。**

---

## 進め方（このフォルダで `claude` が起動したら）

1. まず `ブランド設計プロンプト.md` をユーザーに開いてもらい、**自分のブランドに合わせて書き換えて**もらう（または口頭・チャットで答えてもらう）。
2. 受け取った内容をもとに、`index.html` の中の **`▼▼▼ CONFIG ▼▼▼` 〜 `▲▲▲ CONFIGここまで ▲▲▲`** の間だけを書き換える。
3. 書き換えたら、ローカル確認とデプロイ手順（下記）を案内する。

---

## 絶対ルール

- ✅ **CONFIG ブロックだけ**を編集する。
- 🚫 CONFIG より下（`applyConfig()` 以降＝3Dエンジン、シェーダ、粒子、スクロール制御）は**絶対に触らない**。動かなくなる。
- 🚫 勝手に新セクション・新機能・装飾を足さない（頼まれた範囲だけ）。
- 🚫 事実でない実績・数字を書かない（本人が言った本当のことだけ。盛らない）。

## CONFIG の各項目

| キー | 中身 | メモ |
|---|---|---|
| `brand` | FVの立体文字・ヘッダー・フッター・タブ名 | **英字・短めが映える**（例 NOVA, Sukeads）。日本語も可だが3D文字が細くなる |
| `tagline` | 左上の小さいコピー | 一言 |
| `hero.sub` | FV下のサブ文 | 1行。ブランドの約束を短く |
| `chapters[]` | 4章（順に：痛み→再定義→解決→証拠）。`title`(英大文字)・`idx`・`label`・`accent`(色)・`body`(本文, 改行は`<br>`) | 下の「コピーの質」を厳守 |
| `join.heading` | 最終画面の見出し（改行`<br>`） | 例 `JOIN THE<br>SUKEADS` |
| `join.font` | 見出しフォント | 変えたら `<head>` の Google Fonts `<link>` にも足す（下記） |
| `join.body` | 最終画面の本文 | |
| `join.ctaText` / `join.ctaUrl` | CTAボタンの文言 / リンク先 | ctaUrl は LINE友だち追加・申込URL等。`'#'`にすると押せないボタンになる |
| `footer` | フッター文 | HTMLタグ可 |
| `background` | 背景モード | `'grid-all'`(全章サイバーグリッド) / `'grid-hero'`(FVだけグリッド→以降星雲) / `'nebula'`(星雲のみ) |
| `tints` | 各章の色 `[背景, アクセント]` を6組（16進 `0x......`） | 順に HERO / CONNECTION / ALGORITHM / IGNITION / VOICE / JOIN |

## コピーの質（ここが一番大事）

- **説明しない。短く言い切る。** LPは詩とスローガンの中間。1〜2行で刺す。段落で説明するとダサい。
- **終止形ベース**。体言止め・「〜なんだ」は避ける。命令形の挑発（例「〜しろ」「〜するな」）は世界観に合えば可。
- **煽り単体・誇張NG**。価値と設計で"動きたくなる"状態を作る。CTAも「今すぐ登録！」ではなく価値提示で。
- **数字・実績は本当のことだけ**。無ければ書かない。
- 4章は説得の流れ（①痛みの自覚 ②常識の再定義 ③解決＝自分にもできる ④証拠/選択）を意識。ユーザーが自分の言葉を持っていればそれを尊重し、無ければこの流れで提案する。

## フォントを変えるとき

`join.font` を変更したら、`index.html` の `<head>` にある Google Fonts の `<link>` にそのフォント名を追加する（例 `&family=Orbitron:wght@700`）。追加しないと反映されない。

## 確認とデプロイ（ユーザーに案内する）

- **ローカル確認**：このフォルダで `python3 -m http.server 8803` → ブラウザで `http://localhost:8803/`（更新は必ず ⌘+Shift+R＝キャッシュ回避）。
  - ※WebGLはタブが非アクティブだと止まる。動きを見る時はタブを前面に。
- **公開（無料）**：`npx wrangler login`（初回、ブラウザで承認）→ `npx wrangler pages deploy . --project-name <好きな名前>` → `https://<名前>.pages.dev` で公開。
  - もしくは `dash.cloudflare.com` → Workers & Pages → Pages → このフォルダをドラッグ&ドロップ。

## 依存

three.js / GSAP / Lenis / Google Fonts はすべてCDN読込。ネット接続があればそのまま動く。ビルド不要・1ファイル完結。
