# ブランド設計プロンプト（これを自分用に書き換えて Claude Code に渡す）

使い方：下の【　】を自分のブランドに置き換える。分からない項目は「お任せ」でOK。
書けたら、**この 1〜10 全部をコピーして Claude Code に貼り付ける**（このフォルダで `claude` を起動した状態で）。Claude が `index.html` の CONFIG を書き換えて、あなた専用のLPを作る。

---

これから伝えるブランド情報をもとに、このフォルダの `index.html` の **CONFIG ブロックだけ** を書き換えて、私専用のランディングページを作って。CONFIGより下（3Dエンジン）は触らないで。

1. **ブランド名（FVの大きな立体文字になる）**：【 例：NOVA 】
   ※英字・短めが一番映える。日本語でも可。

2. **これは何のLP？誰向け？（一言で）**：【 例：AIで最短でスキルを身につけたい社会人向けの学習コミュニティ 】

3. **ターゲットの悩み・不安（本音のところ）**：【 例：教材は買うのに続かない。何から手をつければいいか分からない 】

4. **提供する価値・オファー（無料プレゼント等）**：【 例：AI学習ロードマップを無料配布。LINEで受け取れる 】

5. **CTAボタン**：
   - 文言：【 例：無料でロードマップを受け取る 】
   - リンク先URL：【 例：https://line.me/R/ti/p/@xxxxx 】

6. **トーン（どんな空気感か）**：【 例：挑発的で熱い／上品で知的／やさしく寄り添う … から近いもの 】

7. **世界観の色**：【 例：青×シアン／紫／炎のオレンジ／緑 … 】

8. **背景モード**：【 grid-all（全章サイバーグリッド）／ grid-hero（最初だけグリッド→星雲）／ nebula（星雲のみ）から1つ。迷ったら grid-all 】

9. **各章で言いたいこと**（自分の言葉があれば。無ければ「お任せ」）：
   - 01 痛み（読者の"刺さる自覚"）：【 　】
   - 02 再定義（常識をひっくり返す一言）：【 　】
   - 03 解決（"自分にもできる"の理由）：【 　】
   - 04 証拠 or 選択（実績 or 二択の問い）：【 　】
   ※実績・数字を書くなら本当のことだけ。無ければ書かなくていい。

10. **最終画面(JOIN)の見出し**：【 例：JOIN THE NOVA 】

---

**コピーの作り方（守ってほしい）**：
- 説明しないで短く言い切る。1〜2行で刺す。段落で説明しない。
- 終止形ベース。煽り・誇張・盛りはしない。価値で動かす。
- 事実だけ。数字や実績は本当のことだけ書く。

書き換えたら、ローカル確認の手順（`python3 -m http.server` で開く／⌘+Shift+R）と、公開の手順（Cloudflare Pages）も教えて。
