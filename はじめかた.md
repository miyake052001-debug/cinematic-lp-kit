# はじめかた（これを貼るだけ）

このキットは、**あなたが質問に答えるだけ**でシネマティックなホームページ（LP）ができます。
ネットやデザインの知識はいりません。むずかしい入力欄もありません。
（※表示にはインターネット接続が必要です）

## やること（3ステップ）

1. このフォルダで Claude Code を起動する：
   ```
   claude
   ```
2. **`/start` と打つ**。（もし出てこなければ、下の枠の文をコピペするか、ただ「**サイトを作りたい**」と話しかけるだけでもOK）
3. あとは、聞かれたことに思いつくまま答えるだけ。分からなければ「**おまかせ**」と言えばOK。

---

（↓ `/start` が使えない時にコピーして貼る）

> このキットでホームページ（LP）を作りたいです。ネットやデザインは詳しくないので、専門用語を使わずに、一つずつやさしく質問してください。おすすめや例も出してくれると助かります。準備ができたら、最初の質問をどうぞ。

---

## このあとの流れ

- Claude が **7つくらいの質問を一つずつ**してくれます。
  （どんなサイト？ / 誰に届けたい？ / 何をあげる？ / 最後に何をしてほしい？ / 大きく出す名前は？ / 雰囲気と色は？ / 背景の見せ方は？）
- 背景は **①きれいな写真がゆっくり動くタイプ ②光の粒が形を変えるタイプ** から選べます（迷ったら①）。
  写真タイプなら、あなたの写真を `assets/photos/` に入れるか、Claudeが写真づくり用の呪文（プロンプト）を渡します。
- あなたが答えると、Claude が **文章もデザインも全部作って**、あなた専用のLPになります。
- できあがったら、その場で確認 → **無料で公開**（やり方も Claude が教えてくれます）。

気楽に、思ったことを話すだけで大丈夫です。
